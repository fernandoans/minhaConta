# Minha Conta

Desenvolvido com o [Sails v1](https://sailsjs.com)

Dashboard para controle de contas pessoais.

Permite uma visão gerencial da conta por parte do cliente.

## Tecnologias Usadas

+ Banco MySQL
+ Node.js
+ NPM
+ Sails.js
+ Highcharts.js
